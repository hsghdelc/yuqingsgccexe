# 舆情筛查系统

> 自动化筛查招聘和薪酬管理相关负面舆情信息。
> 在线地址：sgccyuqing.ccwu.cc

## 功能特性

### 核心识别引擎
- **三层匹配机制**：实体词 + 领域词 + 负面情感词 AND 匹配，降低误报率
- **实体词分层**：强实体（国网/具名子公司）出现即命中，弱实体（供电局/省电力）须与领域词邻近共现
- **AI 二次核验**：集成 DeepSeek 等大语言模型，对本地命中结果进行二次确认
- **三色关键词高亮**：导出 Excel 中实体词（蓝）、领域词（紫）、情感词（红）分色标注

### 内容解析（桌面版）
- 从 Excel 中的 URL 自动提取网页文字内容
- 三层递进抓取：http 快速提取 → BrowserWindow 渲染 → 弹窗登录
- 支持微博、今日头条、抖音、知乎等平台前置登录，Cookie 自动持久化

### 词库逆向（桌面版）
- 通过 Dify 智能体从舆情数据中自动提取高价值词根
- 输出关键词、负面频次、风险得分、词汇分类等分析报表

### 词库管理
- 实体词库（蓝色）、领域词库（紫色）、负面情感词库（红色）
- 强弱实体实时高亮（绿色=强，黄色=弱）
- 词库版本检测，更新时自动重置

## 技术栈

| 层级 | 技术 |
|---|---|
| 前端 | HTML/CSS/JavaScript（单文件架构） |
| Excel 读取 | SheetJS |
| Excel 写入 | ExcelJS（支持富文本和样式） |
| AI 接口 | OpenAI 兼容 API |
| 智能体 | Dify Workflow |
| 网页抓取 | Node.js + cheerio + Electron BrowserWindow |
| 桌面打包 | Electron + electron-builder |

## 使用流程

1. **导入**：上传包含舆情数据的 Excel 文件
2. **配置**：设置识别规则、词库、AI 模型等参数
3. **扫描**：运行本地识别或 AI 辅助识别
4. **导出**：导出带关键词高亮标注的 Excel 结果
